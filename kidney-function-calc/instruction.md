# Creatinine Clearance (Cockcroft-Gault)

A 68-year-old woman weighs 60 kg. Her serum creatinine is 1.2 mg/dL.

Estimate her creatinine clearance using the Cockcroft-Gault equation.

Write **only** the numeric result in mL/min, rounded to one decimal place,
to the file `/app/answer.txt`. Do not include units or any other text.
